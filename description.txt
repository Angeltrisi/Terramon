The Terramon Mod is a Pokémon-inspired world in Terraria. Now remastered for 1.4!

Features:
- Over 145 Pokémon to find and catch, including their Shiny forms
- 4 different Poké Balls with interactive mechanics
- A brand new Town NPC, the Poké Mart 
- Custom user interface (UI) elements
- Vanity and other decorative items

Note: Pokémon battling is coming in version 1.0!

The source code of the mod is available at
https://github.com/nsfury/Terramon

Make sure to check out the mod's homepage for additional information.
